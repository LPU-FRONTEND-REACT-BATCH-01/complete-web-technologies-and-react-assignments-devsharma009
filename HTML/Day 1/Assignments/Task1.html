Mathematical Formulas

Here are some mathematical formulas:

1. Pythagorean theorem : P2 + B2 = H2

2. Quadratic formula : x = (-b ± √(b2 - 4ac)) / 2a

3. Area of a circle : A = πr2

4. Perimeter of a rectangle : P = 2(l + w)

5. Area of a rectangle : A = l x w

6. Area of a triangle : A = (1/2) x b x h

7. Circumference of a circle : C = 2πr

8. Simple interest : SI = (P x R x T) / 100

9. Average formula : Average = Sumvalues / n

10. Speed formula : Speed = Distance / Time


Chemical Reactions

Here are some chemical reactions:

1. Combustion of methane : CH4 + 2O2 → CO2 + 2H2O

2. Photosynthesis : 6CO2 + 6H2O → C6H12O6 + 6O2

3. Neutralization reaction : HCl + NaOH → NaCl + H2O

4. Decomposition of hydrogen peroxide : 2H2O2 → 2H2O + O2

5. Rusting of iron : 4Fe + 3O2 + 6H2O → 4Fe(OH)3

6. Formation of water : 2H2 + O2 → 2H2O

7. Carbon dioxide formation : C + O2 → CO2

8. Sodium reacting with water : 2Na + 2H2O → 2NaOH + H2

9. Formation of ammonia : N2 + 3H2 → 2NH3

10. Decomposition of calcium carbonate : CaCO3 → CaO + CO2
